Назначение переменных в .env:
    1. PHONE - телефон, привязанный к аккаунту телеграмма
    2. API_ID - ID из конфигурации приложения
    3. API_HASH - HASH из конфигурации приложения
    4. SAVE_FOLDER - папка сохранения результатов
    5. MESSAGES_AGE - возраст сообщений (в днях)
    6. MONITORED_CHATS - ID чатов, откуда мониторить сообщения (вы должны состоять в этих чатах)


Получение API_ID и API_HASH:
    1. Переходим по ссылке на сайт телеграмм https://my.telegram.org/auth и вводим свой телефон
    2. Приходит код, который нужно вставить и нажать зарегистрироваться
    3. Дальше выбираем API development tools и нажимаем продолжить (https://my.telegram.org/apps)
    4. После регистрации мы попадаем в окно с API_HASH и API_ID, которые нам надо скопировать для работы бота


Как узнать ID любого канала/чата в телеграм:
    1. Открываем любой чат в браузере с компьютера
    2. Смотрим на URL и после https://web.telegram.org/a/# будет написан ID чата
    3. Например для https://web.telegram.org/a/#-1001963653498 ID чата будет -1001963653498
    4. Учтите, что для работы программы вы должны быть подписаны на канал, который парсится
